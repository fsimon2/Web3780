Key terms from chapter two

1. Description lists
This html tag is used for terms and definitions. w3schools

2. Ordered lists
This tag is used to order stuff in a list by adding a number to it. w3schools

3. Block-level elements
Block level elements always start on a new line and will always take up the full width available w3schools

4. External Style Sheet
An external style sheet is a separate CSS file linked to an HTML document to apply consistent styles across multiple pages Source

5. Hexadecimal Color
Hexadecimal color codes are used in HTML and CSS to specify colors. They are written as #RRGGBB, where RR, GG, and BB are hexadecimal values fix this

6. Comment
In HTML, comments are used to insert notes into the source code. They are not displayed in the browser. w3schools

7. Attribute Name
The name attribute specifies a name for an HTML element. This name attribute can be used to reference the element in a JavaScript w3schools

4. Class Selector
In CSS, a class selector is used to select elements with a specific class attribute. It is defined with a period (.) followed by the class name w3schools

9. Inline Style
Inline styles are CSS styles that are applied directly within an HTML element using the style attribute w3schools

10. Void Tag
A void tag is an HTML tag that does not have a closing tag. w3schools
